The plugin disables ObjectHook and sets Aim to Game during a cutscene.


How to use:

1\) Download & copy CutsceneDetectionPlugin.dll file to
```
%AppData%\UnrealVRMod\{GAME_EXE_NAME}\plugins
```
2\) Enable load ObjectHook at startup via UI or in %AppData%\UnrealVRMod\{GAME_EXE_NAME}\config.txt file (UObjectHook_EnabledAtStartup=true)
